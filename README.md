android_device_lge_lu3700
=========================

CyanogenMod10 device tree for IM-A840S (SKY Vega S5) - Jellybean Branch

Working:
* RIL
* Graphics
* Sound
* Wifi
* Tethering
* Bluetooth
* Sensors
* GPS
* Camera
* Video

Known issues:
* Bluetooth SCO is not working
* HDMI is not tested
* Properiatary softwares are not working ( DMB, etc...)
